
BET33 프런트(정적) v1 — 배포 안내
=================================

1) 파일 구조
- index.html
- styles.css
- app.js
- config.js  ← 카카오/텔레/가입 URL을 @@@@@플레이스홀더@@@@@로 제공. 값만 교체하세요.
- /assets/
   ├─ logo.png (제공된 로고 복사)
   ├─ emergency-bg.mp4 (미리보기용 샘플, 실서비스에서 제거 가능)
   └─ /banners/ (선택)

2) 동영상 자산(사용자 제공, 파일명 고정 권장)
- /assets/쿠폰신청하기.mp4
- /assets/긴급쿠폰신청.mp4

3) 동작 요약
- 헤더 버튼
  • 프로모션: 배경 스윕(상 금빛 좌→우 / 하 파랑 우→좌)
  • 쿠폰신청: 중앙 모니터 재생 → 0.25s 후 전체 배경 재생 → 0.25s 후 중앙 복귀 + 작은 폼(ID, 24시간내 이용횟수) 표시 → [확인]으로 종료
  • 프로모션참여: 4종 카테고리 선택 → [확인] 시 “회원만 참여” 안내 후 1s 뒤 긴급쿠폰신청.mp4 재생
  • 문의: 카카오톡/텔레그램 링크 팝업 (config.js 수정)
  • 긴급쿠폰신청: 긴급쿠폰신청.mp4 재생(모니터→배경→종료)
  • 가입하기: 가입 폼 → 축하 팝업 → 3지선다 체크 설문 → 쿠폰받기

4) 그리드/치수
- 전체 폭: 33.88cm (±1cm 허용)
- 헤더 높이: 6.25cm
- 중앙 모니터: 가로 19cm × 세로 10.8cm
- 좌/우 패널: 좌 8cm / 우 6cm (PPT 범위 내에서 균형 배치)
- 상단/하단 라인 두께: 0.28cm
- 섹션 간격: 1.66cm
- 인라인 폼: 가로 10cm × 높이 1.3cm

[Tip] 키보드 G 를 눌러 1cm 간격 보조그리드를 표시/숨김 할 수 있습니다.

5) 배포
- GitHub Pages: 리포지토리 루트에 본 파일들을 올리면 즉시 반영됩니다.
- 정적 호스팅: Nginx/Apache 등 어디든 그대로 사용 가능합니다.
